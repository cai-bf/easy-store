# easy-store



## QuickStart

### initial
```bash
$ cp .env.example .env
// edit .env, fill cinfig
$ cp database/config.example.json database/config.json
// edit database/config.json, fill config
$ npx sequelize db:migrate
```

### Development

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

### Deploy

```bash
$ npm start
$ npm stop
```
